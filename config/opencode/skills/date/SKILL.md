# Date Skill

Get the current date using the bash `date` command.

## When to Use
- Any time you need the current date
- Avoid date inference

## Command

```bash
date
```

## Common Pitfalls
- Inferring the date from context
- Hard-coding dates without checking

## Example

```bash
date "+%Y-%m-%d"
```
