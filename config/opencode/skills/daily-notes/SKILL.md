The `daily-notes` agent is the source of truth for all daily notes work.

If the user is asking to update, plan, review, or summarize their daily notes, direct them to open the `daily-notes` agent (via `ctrl+p` → Agents → daily-notes), or switch to it directly.

Do not attempt to execute the daily notes workflow inline — the agent has the correct permissions, model settings, and full workflow context.
